# Внедрение фотографий и улучшение UI/UX для PLANTOS

Сайт PLANTOS — отличный проект с минималистичным дизайном, но сейчас ему не хватает визуальной "сочности" и динамики. Я изучил код (React + Vite) и подготовил план того, как мы сделаем его премиальным и живым.

## User Review Required

> [!IMPORTANT]
> Я собираюсь сгенерировать 12 красивых реалистичных фотографий для каждого растения с помощью нейросети и положить их в папку `public/`. Согласны ли вы на такой подход, или вы хотите использовать свои собственные фотографии?

## Proposed Changes

### 1. Добавление фотографий
Сейчас вместо фото используются векторные контуры листьев.
- **Генерация фото**: Я сгенерирую 12 качественных фотореалистичных изображений для каждого растения из `catalog.js` (Монстера, Замиокулькас, Фикус Лирата и т.д.) в едином эстетичном стиле.
- **Интеграция**:
  - Сохраню файлы в `public/plants/`.
  - Обновлю `catalog.js`, добавив поле `image` каждому растению.
  - Обновлю `App.jsx`, чтобы в карточках `<PlantCard>` и модалке `<ProductModal>` выводились эти фотографии вместо иконок, с красивым заполнением (`object-fit: cover`).

### 2. Микро-анимации (Micro-interactions)
Чтобы сайт ощущался "живым", я добавлю следующие анимации в `index.html` (CSS):
- **Плавное появление каталога (Staggered Fade-In)**: При загрузке страницы карточки будут мягко всплывать одна за другой.
- **Hover-эффекты карточек**: При наведении карточка будет слегка приподниматься (`transform: translateY(-4px)`), а фотография внутри плавно увеличиваться (zoom-in).
- **Анимация корзины**: При добавлении товара кнопка корзины (в шапке) будет делать легкий "pop" (отскок), привлекая внимание.
- **Смягчение модальных окон**: Более плавный и кинематографичный выезд боковой панели и появление модалки товара.

### 3. Мои предложения по UX/UI (Фишки)
- **Эффект матового стекла (Glassmorphism)**: Сделаем верхнюю панель (`topbar`) и боковой фильтр (`sidebar`) слегка полупрозрачными с эффектом размытия фона (`backdrop-filter: blur(12px)`). При прокрутке каталога карточки будут красиво просвечивать под шапкой.
- **Очистка фильтров**: Добавить анимацию тряски (shake) кнопке "сбросить", если пользователь пытается сбросить пустые фильтры.
- **Скелетная загрузка (Skeleton Loading)**: Пока изображения подгружаются, будет отображаться пульсирующий серый фон, чтобы интерфейс не прыгал.
